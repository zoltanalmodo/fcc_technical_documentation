    GIT

    using git / terminal commands

    when create a new project we need to install GIT inside the project folder.
    Use terminal$ git init (enter) to install git inside the project folder.
    this will create a file for git where all further history of the project files will be recorded.

    how to create a new repository on the command line?


    echo "# test_project" >> README.md
    git init
    git add README.md
    git add . (adds all files to the staging area)
    git commit -m "first commit" (make a commit and leave a comment)
    git remote add origin https://github.com/zoltanalmodo/test_project.git
    (points to repository after enabling github pages) git push -u origin master (pushes all to master)

    push an existing repository from the command line

    git remote add origin https://github.com/zoltanalmodo/test_project.git
    git push -u origin master
